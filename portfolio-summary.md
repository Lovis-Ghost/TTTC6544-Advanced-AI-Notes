# Portfolio Summary

This repository is my personal learning portfolio for TC6544 Advanced Artificial Intelligence. I use it to keep bilingual notes, small reflections, and simple Python demos from the topics covered in class.

The project currently covers AI optimization basics, state space search, heuristic search and A*, local search, simulated annealing, great deluge, tabu search, genetic algorithm, and hyper-heuristics. For topics not covered yet, I only created placeholder notes so I can return to them later.

The Python demos include BFS/DFS, A* search, greedy TSP nearest neighbour, hill climbing for TSP, simulated annealing, great deluge, tabu search, and a Genetic Algorithm for the MAXONE problem.

From building this repo, I learned that many AI algorithms are connected by the same basic idea: search through possible solutions and decide what to try next. I also got more comfortable explaining terms like objective function, heuristic, neighbour, fitness, and local optimum in my own words.

Next, I want to improve my hand-calculation practice and add better notes after the remaining lectures, especially nature-inspired algorithms, neural networks, and fuzzy logic.
